# Dual Game - Technical Demo

¡Bienvenido a **Dual Game**! Esta es una demo técnica desarrollada para explorar y poner en práctica las capacidades de maquetación web moderna.
El proyecto consiste en la creación de una página web plana (estática) diseñada desde cero. El objetivo principal es perfeccionar las habilidades de estructuración y diseño visual, simulando la interfaz de una plataforma de comercio electrónico.

---

## Tecnologías Empleadas
Este proyecto se centra en el uso de estándares web nativos para garantizar un código limpio y eficiente:

* **HTML5:** Estructura semántica y organización de contenido.
* **CSS3:** Estilos avanzados, animaciones y diseño adaptable (Responsive).

---

## Objetivos de la Práctica
* **Semántica HTML:** Uso correcto de etiquetas para accesibilidad y SEO.
* **Maquetación:** Implementación de layouts complejos mediante CSS.
* **Estética:** Creación de una interfaz de usuario atractiva y funcional para el sector gaming/e-commerce.

---

## Estructura del Proyecto
El repositorio está organizado de la siguiente manera:
* `paginaPrincipal.html`: Estructura principal de la web.
* `/Material`: Guarda el favicon y otros elementos tecnicos
* `/img`: Carpeta de recursos gráficos y multimedia.

---

## Instalación y Uso
Para ver el proyecto en tu máquina local, sigue estos pasos:

1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/AdriaGar/Projecte-ecomerce-DAM.git](https://github.com/AdriaGar/Projecte-ecomerce-DAM.git)
    ```
2.  **Abre el archivo principal:**
    Navega a la carpeta del proyecto y abre `paginaPrincipal.html` en cualquier navegador moderno.
